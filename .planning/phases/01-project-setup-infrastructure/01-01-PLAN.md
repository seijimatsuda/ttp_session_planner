  <action>
Install Tailwind CSS v4 with the Vite plugin:

```bash
cd /Users/seijimatsuda/session_planner_1-22-26/frontend
npm install tailwindcss @tailwindcss/vite
```

Update vite.config.ts to include the Tailwind plugin:

```typescript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(),
  ],
})
```

Replace src/index.css content with Tailwind import (this is Tailwind v4 syntax - no directives):

```css
@import "tailwindcss";
```

Ensure main.tsx imports index.css (should already be there from scaffold).

IMPORTANT: Tailwind v4 does NOT need:
- tailwind.config.js (auto-detects content)
- postcss.config.js (uses Vite plugin)
- @tailwind directives (uses @import)
  </action>

```bash
grep -q "@import" src/index.css && echo "CSS import configured"
```
  </verify>
  <done>Tailwind CSS v4 installed and configured with Vite plugin</done>
